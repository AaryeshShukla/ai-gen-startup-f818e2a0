# TaskFlow AI Landing Page

## Overview
TaskFlow AI is a modern SaaS landing page designed for an AI-powered workspace and automation engine. The page features a dark theme with neon gradients, interactive components, and a clean layout.

## Features
- Sticky navigation bar with a glowing CTA button.
- Animated hero section with an email capture form.
- Interactive feature cards that scale and glow on hover.
- Dynamic pricing toggle between monthly and annual plans.
- Interactive FAQ accordion for user inquiries.
- Customer testimonial slider.
- Elegant footer with social links.

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts (Outfit, Inter)

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Enjoy the interactive features and modern design!
